Bank Management system

The Bank Management System is a comprehensive application designed to manage user accounts, transactions, and banking services efficiently. Users can deposit and withdraw money, check account details, pay bills, and update passwords. Admins have control over account management, including adding, deleting, and searching accounts. The system also offers essential banking services like loan applications, currency exchange, and complaint registration. A secure OTP-based authentication system ensures guest users can log in safely, with a built-in timer and loading animation enhancing the user experience.

Hangman Game

The Hangman game is a word-guessing challenge where players attempt to reveal a hidden word by guessing one letter at a time. The game selects a random word from categories like animals, fruits, vegetables, and objects. With each incorrect guess, the Hangman figure progresses step by step in an ASCII representation. Players have a total of six chances to guess the correct word before losing. The program also enhances user experience by incorporating color changes for visual appeal, making the gameplay more engaging.

Word Hunt

The Word Hunt game is a grid-based word search where players find hidden words within a randomized letter grid. Players can choose from three difficulty levels—Easy, Medium, and Hard—each increasing the grid size and the number of words to find. The game randomly selects words and places them strategically within the grid while filling the remaining spaces with random letters. Players have a limited time to identify words, and a built-in timer creates urgency. Once the time expires, the game evaluates performance and displays whether the player wins or loses.

Matrix-Based Encryption & Decryption System

This program implements a matrix-based encryption and decryption system that secures messages using matrix multiplication. A random 3x3 key matrix is generated to encode the message, which is then converted into a numerical matrix and encrypted through multiplication with the key matrix. To decrypt, the system calculates the inverse of the key matrix and applies it to the encrypted matrix, restoring the original message. The program supports manual input and file handling, ensuring flexibility in message encryption. Additionally, it features error handling, loading animations, and a colorful interface to enhance user experience while maintaining data security.

Game Hub

Assembly Performance Analyzer

This Assembly Language program functions as both a calculator and a system performance analyzer. It includes a number system converter, allowing users to convert values between decimal, binary, octal, and hexadecimal. Additionally, it supports matrix operations, including addition and multiplication, with an execution time tracker that measures the speed of these calculations. Beyond mathematical functions, the program also benchmarks system performance by analyzing file operations such as read, write, and copy times, along with computing turnaround time to assess CPU efficiency. By leveraging low-level assembly instructions, this program provides accurate calculations and real-time execution insights, making it a valuable tool for both computation and benchmarking.

Text Typing Speed Tester

This Assembly Language program measures the typing speed of the user by tracking the time taken to type a given text. The program starts a timer when the user begins typing and stops it upon completion, calculating the words per minute (WPM) and overall accuracy. Additionally, it evaluates typing efficiency by analyzing input delay and keystroke speed. Using low-level hardware interaction, the program ensures precise time measurement, making it an effective tool for benchmarking typing speed in a minimal and optimized environment.
